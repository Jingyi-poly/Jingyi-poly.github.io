# Images Directory

This directory contains images for your personal website.

## Required Image

Please add your profile photo to this directory:

- **Filename**: `bio-photo.jpg`
- **Recommended size**: 400x400 pixels or larger (square format)
- **Format**: JPG, PNG, or other web-friendly formats

## How to Add Your Photo

1. Prepare your photo (square format recommended)
2. Rename it to `bio-photo.jpg`
3. Place it in this directory (`assets/images/`)

The photo will automatically appear in:
- The sidebar on all pages
- Your author profile section

## Optional Images

You can also add other images for:
- Publications illustrations
- Project screenshots
- Conference presentations
- Any other visual content

Simply place them in this directory and reference them in your markdown files using:

```markdown
![Alt text](/assets/images/your-image.jpg)
```
